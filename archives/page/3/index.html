用模式字母实现日期格式化输出 Jan 29 2015

Java在SimpleDateFormat类中使用模式字母实现日期的格式化输出。


日期和时间模式
日期和时间格式由日期和时间模式 字符串指定。在日期和时间模式字符串中，未加引号的字母‘A’ 到‘Z’ 和‘a’ 到‘z’ 被解释为模式字母，用来表示日期或时间字符串元素。文本可以使用单引号 (‘) 引起来，以免进行解释。"’’" 表示单引号。所有其他字符均不解释；只是在格式化时将它们简单复制到输出字符串，或者在解析时与输入字符串进行匹配。

定义了以下模式字母（所有其他字符‘A’ 到‘Z’ 和‘a’ 到‘z’ 都被保留）：

字母日期或时间元素表示示例

w年中的周数Number27
W月份中的周数Number2
D年中的天数Number189
d月份中的天数Number10
F月份中的星期Number2
E星期中的天数TextTuesday;Tue
aAm/pm 标记TextPM
H一天中的小时数（0-23）Number0

ham/pm 中的小时数（1-12）Number12

Z时区RFC 822 time zone-0800

浅谈MVC Jan 18 2015

对于初学者来说，会有点不懂什么是MVC，为什么要用MVC，不过如果从编程代码进化的角度来看，一切就变得容易了：

1.PHP本身是一种嵌入式脚本语言，所以最初是HTML代码与PHP代码混编的。
2.随着项目变大，代码出现分工，以便于维护和管理。HTML代码和PHP代码分别由两部分人负责，出现了显示和逻辑相分离。
3.随着项目变大，数据操作逻辑的代码经常重复，为便于重用，于是将处理数据的代码从php代码中分离,那么就分成了处理业务逻辑和控制整体流程的两部分代码。

SQL语句中字符串前加N解决中文乱码 Jan 17 2015

之前给Oracle数据库插入数据时，插入中文字符串后都变成乱码了。解决方法是在字符串前加大写的N。

加N表示字符串强制用 Unicode 方式存储。

Unicode 字符串的格式与普通字符串相似，但它前面有一个 N 标识符（N 代表 SQL-92 标准中的国际语言 (National Language)）。N 前缀必须是大写字母。

Unicode是双字节字符。对于西文字符，用一个字节来存储过足够了，对于东方文字字符，就需要两个字节来存储。Unicode为了统一、规范、方便、兼容，就规定西文字符也用两个字节来存储。

微软也有相关的帮助文档：当您在 SQL Server 中的 Unicode 字符串常量处理，则必须在前面带有前缀 N 的所有 Unicode 字符串

当然这里的解决方法除了在字符串前加N，也可以通过更改数据库的字符集来解决。

MyEclipse常用快捷键 Jan 7 2015

MyEclipse或Eclipse编写Java程序时的常用快捷键：

Ctrl+1：快速修复键，万能键，提供错误修复建议
Alt+/：内容提示键
Ctrl+Shift+o：包整理键
Ctrl+Shift+F：自动格式化代码（但格式化后可能会与公司要求格式不同）
Ctrl+Shift+/：添加（除去）块注释
Ctrl+Alt+↑或↓：复制行到上面或下面
Alt+Shift+M：提取方法
