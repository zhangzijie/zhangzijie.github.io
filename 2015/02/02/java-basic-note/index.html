1.main函数写错

2.byte的取值范围

这里byte不会出现因转换丢失精度产生编译错误，只会出现高位丢失，byte取值范围是0-127，所以在这里b取了4096低8位的值，即0。

3.byte的类型提升

byte b = b1 + b2;这句编译错误：不能从int转换成byte。因为两个byte型数字相加时会自动提升为int型。

4.print的输出类型

这里输出的是9.0而不是9，因为99.9是浮点型，表达式结果的最终数据类型，由表达式中出现的最高的数据类型决定。

5.+运算符

这里的输出结果是3java34，因为字符串的+号连接运算符优先级高于算术运算的+号运算符，+运算符是连接符还是算术运算符取决于它左边的操作数。
