CSS3编写3D立方体动画

20行CSS3代码即可构建一个三维立方体以及它旋转的动画。

先说CSS3的几个知识点：
1.transform
transform 属性向元素应用 2D 或 3D 转换。该属性允许我们对元素进行旋转、缩放、移动或倾斜。详见：W3School 3D转换

2.@keyframes
@keyframes 规则用于创建动画。在 @keyframes 中规定某项 CSS 样式，就能创建由当前样式逐渐改为新样式的动画效果。详见：W3School 3D动画

了解了这两个知识点下的相关的属性后，在技术上已经学完了。剩下关键的就是创建立方体的思路了。
立方体通过包含6个li标签的一个ul来承载，一个li代表立方体的一面，前四个li代表立方体的前面，上面，后面和下面，第5、6个li代表立方体的左面和右面。
建立右手空间直角坐标系，第1个li于不用旋转，平面延Z轴正方向平移至前面；第2个li延X轴正方向顺时针旋转90度，整个坐标系跟随旋转90度，然后再延旋转后的坐标系的Z轴正方向平移至上面；第3、4个li是同理的，第3个li延X轴正方向顺时针旋转180度，整个坐标系跟随旋转180度，然后再延旋转后的坐标系的Z轴正方向平移至后面；第4个li延X轴正方向顺时针旋转270度，整个坐标系跟随旋转270度，然后再延旋转后的坐标系的Z轴正方向平移至下面；第5个li延Y轴正方形顺时针旋转90度，整个坐标系跟随旋转90度，然后再延旋转后的坐标系的Z轴正方向平移至左面；第6个li延Y轴正方形顺时针旋转270度，整个坐标系跟随旋转270度，然后再延旋转后的坐标系的Z轴正方向平移至右面。那么立方体就构建完成了。
立方体构建出来之后，可以通过分别延X轴和Y轴旋转360度实现立方体旋转展示的动画了。
